## 📍 区分分類（Position Zone）

当選数字が出現する行の位置によって、以下の3つの区分に分類します。

### 区分ルール（排他的）

| 区分 | 英語名 | 条件 | 対象行 |
|------|--------|------|--------|
| **上部** | upper | 1行目を含む | 1,2,3行目 |
| **中部** | middle | 2,3,4行のみ | 2,3,4行目 |
| **下部** | lower | 5行目を含む | 3,4,5行目 |

### 分類の優先順位

1. **1行目を含む** → 上部（upper）
2. **5行目を含む** → 下部（lower）
3. **2,3,4行のみ** → 中部（middle）

### 例

| 位置 | 区分 | 理由 |
|------|------|------|
| (1,2), (2,3), (3,4) | 上部 | 1行目を含む |
| (2,3), (3,4), (4,5) | 中部 | 2,3,4行のみ |
| (3,4), (4,5), (5,6) | 下部 | 5行目を含む |

## 1. 前提条件

### 1.1 分析対象

- **対象**: 極CUBEの1〜5行目
- **区分**: 上部（1,2,3行）/ 中部（2,3,4行）/ 下部（3,4,5行）
- **当選数字**: N3の3桁（必ず3桁）
- **前提**: 3桁は必ず横・斜め・縦のいずれかでつながっている

### 1.2 つながりの絶対条件

**重要**: 以下の条件を満たさない配置はNG

- ❌ 連続しない行に出現（例：2行目と4行目だけで3行目を飛ばす）のはNG
- ❌ 連続しない列に出現（例：3列と5列だけで4列を飛ばす）のはNG
- ✅ 必ず横・斜め・縦のいずれかで連続してつながっている必要がある

### 1.3 並び型判定の汎用化

並び型の判定は**相対位置**で行います。これにより：
- 上部（1,2,3行）、中部（2,3,4行）、下部（3,4,5行）のどの区分でも同じロジックで判定可能
- 「連続する3行」という条件で判定し、具体的な行番号に依存しない

### 1.3 極CUBEの構造

- **行数**: 最大5行（メイン行は1,3,5行目）
- **列数**: 固定8列（1-8列）
- **分析対象行**: 2行目、3行目、4行目のみ

## 2. つながりの判定ロジック

### 2.1 横方向のつながり

**定義**: 同じ行で**隣接する列**に数字が存在する

**例**:
- ✅ 2行目2列と2行目3列 → つながっている
- ❌ 2行目2列と2行目4列 → つながっていない（1列を挟んでいる）

**判定ロジック**:
```python
def is_horizontal_connected(pos1, pos2):
    """
    pos1, pos2: (row, col) のタプル
    """
    if pos1[0] != pos2[0]:  # 同じ行でない
        return False
    return abs(pos1[1] - pos2[1]) == 1  # 列の差が1
```

### 2.2 縦方向のつながり

**定義**: 同じ列で**隣接する行**に数字が存在する

**例**:
- ✅ 2行目3列と3行目3列 → つながっている
- ✅ 3行目3列と4行目3列 → つながっている
- ❌ 2行目3列と4行目3列 → つながっていない（3行目を飛ばしている）

**判定ロジック**:
```python
def is_vertical_connected(pos1, pos2):
    """
    pos1, pos2: (row, col) のタプル
    """
    if pos1[1] != pos2[1]:  # 同じ列でない
        return False
    return abs(pos1[0] - pos2[0]) == 1  # 行の差が1
```

### 2.3 斜め方向のつながり

**定義**: **隣接する行・列**で斜めに数字が存在する

**例**:
- ✅ 2行目2列と3行目3列（右下斜め）→ つながっている
- ✅ 2行目4列と3行目3列（左下斜め）→ つながっている
- ✅ 3行目3列と4行目2列（左上斜め）→ つながっている
- ✅ 3行目3列と4行目4列（右下斜め）→ つながっている

**判定ロジック**:
```python
def is_diagonal_connected(pos1, pos2):
    """
    pos1, pos2: (row, col) のタプル
    """
    row_diff = abs(pos1[0] - pos2[0])
    col_diff = abs(pos1[1] - pos2[1])
    return row_diff == 1 and col_diff == 1  # 行・列の差がともに1
```

### 2.4 つながりの総合判定

**3桁がつながっているかの判定**:
- 3桁のうち、少なくとも2組がつながっている必要がある
- すべての桁が連結している必要がある（孤立した桁がない）
