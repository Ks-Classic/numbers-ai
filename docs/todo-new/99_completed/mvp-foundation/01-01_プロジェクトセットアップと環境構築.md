# 01-01 プロジェクトセットアップと環境構築

## 概要
Next.js 15 + TypeScript + Tailwind CSS v4のプロジェクト基盤を構築し、開発環境を整備する。

## 詳細タスク

- [x] Next.js 15プロジェクトを作成する（create-next-app使用）
  - App Routerを使用する ✓
  - TypeScriptを有効化する ✓
  - ESLint設定を追加する ✓（eslint-config-next使用）

- [x] Tailwind CSS v4を導入して設定する
  - Tailwind CSS v4を導入する ✓（@tailwindcss/postcss使用）
  - globals.cssにTailwindのベーススタイルを追加する ✓
  - PostCSS設定を確認する ✓（postcss.config.mjs）

- [x] shadcn/uiを導入する
  - shadcn/uiの初期化コマンドを実行する ✓
  - components.jsonを設定する ✓
  - 必要なコンポーネントをインストールする ✓
    - button, input, tabs, card, badge, dialog, label, progress, radio-group, select, sonner, switch（12個）

- [x] プロジェクト構造を作成する
  - src/app/ディレクトリ構造を作成する ✓（複数ページ実装済み）
  - src/components/ディレクトリ構造を作成する ✓（ui/, shared/, features/, layouts/）

  - tsconfig.jsonをカスタマイズする ✓（paths設定済み）
