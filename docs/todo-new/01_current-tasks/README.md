### 優先度: 最高 - AI精度改善

**ディレクトリ**: `01_current-tasks/ai-improvement/`

**目的**: AIモデルの精度を向上させ、目標値（AUC-ROC: 0.65以上）を達成する

**主要タスク**:

1. **ベースライン確立**
   - ✅ 1000回分でベースラインモデル作成完了
   - ✅ 4801回分でベースラインモデル作成完了（軸数字予測モデルのみ）
     - AUC-ROC: 0.5135（1000回分: 0.5106とほぼ同等）
     - データ量増加だけでは精度向上が限定的

2. **ハイパーパラメータチューニング**
   - ⏳ 1000回分でのハイパーパラメータチューニング（統合モデル）
   - ⏳ 4801回分でのハイパーパラメータチューニング（統合モデル）

3. **精度評価と判断**
   - ⏳ 両データ範囲の精度を比較
   - ⏳ 条件付きで分割モデル検討（4801回分のみ）

**重要な方針決定**:
- 特徴量エンジニアリング実験はスキップ（全特徴量を使用）
- 1000回分での分割モデルは検討しない（統合モデルのみ）
- 4801回分での分割モデルは条件付き（目標未達かつパターン間の精度差が大きい場合のみ）
