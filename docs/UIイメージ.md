### 主要コンポーネント
1. **大型アクションボタン** (最小タップエリア: 48×48px)
2. **カード型リスト** (履歴・統計表示用)
3. **プログレスバー** (AI分析中表示用)
4. **タブ** (N3/N4、ボックス/ストレート切り替え)
5. **アコーディオン** (軸数字候補の展開/折りたたみ)
6. **モーダルダイアログ** (スコア説明表示用)
7. **手動指定ブロック** (ユーザー指定の軸数字で候補を計算)

## 📋 実装済み機能

### ✅ 実装済み
- ホーム画面（新規予測を開始ボタンのみ）
- 回号設定画面（4桁数字入力、電卓風入力）
- リハーサル数字入力画面（自動フォーカス移動、数字キーボード）
- AI分析中画面（ローディング表示）
- 分析結果表示画面
  - N3/N4切替タブ
  - ボックス/ストレート切替タブ
  - 軸数字候補/総合ランキング切り替え
  - 軸数字候補のアコーディオン表示
  - 当選番号候補のテーブル表示
  - 手動指定軸機能（0-9の数字を入力して計算）
  - スコア説明モーダル
- ナビゲーション（予測のみ）

### ❌ 削除済み機能
- 履歴画面（ナビゲーションから削除）
- 設定画面（ナビゲーションから削除）
- 最終予測生成ボタン（分析結果画面から削除）
- 「出所」情報の表示（UIから削除、コードは保持）
- 「今日の回号」表示（回号設定画面から削除）
- ヒントカード（リハーサル画面から削除）

---

## 🚀 特徴

1. **起動が即座**: ログイン不要で開いてすぐ使える
2. **3ステップで完結**: 回号入力 → リハーサル入力 → 予測結果
3. **認知負荷の軽減**: 不要な情報を排除し、本質に集中
4. **全パターン自動予測**: N3/N4、0なし/0ありを自動で処理
5. **モバイル最適化**: スマホでの使用を前提としたUI/UX
6. **シンプルなナビゲーション**: 「予測」のみで直感的
