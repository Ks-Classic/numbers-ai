### 期待される結果（回号6758）

**前回・前々回のデータ:**
- 前回（6757回）: N3 = **621**（百の位=6、十の位=2、一の位=1）
- 前々回（6756回）: N3 = **458**（百の位=4、十の位=5、一の位=8）

**正しい予測出目:**
- 百の位: 前々回4、前回6 → `"4": { "6": [2, 4, 7] }` → **[2, 4, 7]**
- 十の位: 前々回5、前回2 → `"5": { "2": [2, 6, 8] }` → **[2, 6, 8]**
- 一の位: 前々回8、前回1 → `"8": { "1": [0, 2, 5, 6] }` → **[0, 2, 5, 6]**

**結合してソート:**
```
[2, 4, 7] + [2, 6, 8] + [0, 2, 5, 6]
= [0, 2, 2, 2, 4, 5, 6, 6, 7, 8]
```

これが正しい`sourceDigits`です。
