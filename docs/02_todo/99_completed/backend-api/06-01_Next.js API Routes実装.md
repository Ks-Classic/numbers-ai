# 06-01 Next.js API Routes実装

## 概要
Next.js API Routesで予測実行APIを実装する。

## 詳細タスク

- [ ] API Routeの基本構造を作成する
  - src/app/api/predict/route.tsを作成する
  - POSTメソッドのハンドラーを実装する
  - 基本的なレスポンス構造を実装する

- [ ] リクエストバリデーションを実装する
  - Zodスキーマを作成する
  - roundNumberのバリデーションを実装する
  - n3Rehearsalのバリデーションを実装する
  - n4Rehearsalのバリデーションを実装する

- [ ] 予測表生成処理を呼び出す
  - 回号に基づいて予測表を生成する処理を実装する
  - N3/N4、パターンA/Bの予測表を生成する
  - 生成エラーのハンドリングを実装する

- [ ] AI推論処理を呼び出す
  - Cloud Run APIを呼び出す処理を実装する（Phase 4）
  - または、ローカルAI推論処理を呼び出す（MVP）
  - 特徴量計算から推論までを実行する

- [ ] レスポンス形式を実装する
  - PredictionResult型に従ったレスポンスを実装する
  - N3/N4、ボックス/ストレート、軸数字/組み合わせの構造を実装する
  - generatedAtタイムスタンプを追加する

- [ ] エラーハンドリングを実装する
  - バリデーションエラーの処理を実装する（400 Bad Request）
  - データが見つからない場合のエラー処理を実装する（404 Not Found）
  - モデルエラーの処理を実装する（500 Internal Server Error）
  - その他の内部エラーの処理を実装する

- [ ] ロギングを実装する
  - リクエストログを記録する
  - エラーログを記録する
  - パフォーマンスログを記録する

- [ ] APIテストを実装する
  - ユニットテストを書く
  - 統合テストを書く
  - APIテストを実行して確認する
