# 01-03 予測表生成アルゴリズム実装

**最終更新日時:** 2025-11-04 20:00:00 JST

## 概要
中国罫線理論に基づく予測表生成アルゴリズムを実装する（4パターン: A1/A2/B1/B2に対応）。

**4パターンの定義:**
- **A1**: 欠番補足あり（0〜9全追加）+ 中心0配置なし
- **A2**: 欠番補足あり（0〜9全追加）+ 中心0配置あり
- **B1**: 欠番補足なし（0も含めて、すべて欠番補足しない）+ 中心0配置なし
- **B2**: 欠番補足なし（0も含めて、すべて欠番補足しない）+ 中心0配置あり

**重要な変更（2025-11-04）:**
- メイン行の配置: 奇数行（1, 3, 5, 7行目 = 0-indexedでは1, 3, 5, 7行）に変更
- 裏数字の配置: 偶数行（2, 4, 6, 8行目 = 0-indexedでは2, 4, 6, 8行）に変更
- 7行目（0-indexedでは7行目）は数字があれば配置、なければ不要
