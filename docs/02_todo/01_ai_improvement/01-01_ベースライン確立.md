# 01-01 ベースライン確立

## 概要
AIモデルの精度向上のためのベースラインを確立する。データ範囲を拡大しながらベースラインモデルを作成し、精度の変化を確認する。

## 詳細タスク

### Phase 1: 1000回分でベースラインモデル作成

- [ ] データ前処理を実行する
  - notebooks/01_data_preparation.ipynbを実行する
  - config.pyでTRAIN_SIZE = 1000を設定する
  - train_data_1000.csvが生成されることを確認する（5849回〜6849回、約1,000件）
  - データ範囲が正しく設定されていることを確認する

- [ ] 特徴量エンジニアリングを実行する
  - notebooks/03_feature_engineering.ipynbを実行する
  - 軸数字予測データ（n3_axis_data.pkl, n4_axis_data.pkl）を生成する
  - 組み合わせ予測データ（4モデル分）を生成する
  - 全特徴量セットを使用する（約55次元）
  - 特徴量の次元数を確認する

- [ ] モデル学習を実行する
  - notebooks/04_model_training.ipynbを実行する
  - 軸数字予測モデル（n3_axis.pkl, n4_axis.pkl）を学習する
  - 組み合わせ予測モデル（4モデル）を学習する
  - 学習時間を記録する（約2〜3時間）

- [ ] モデル評価を実施する
  - AUC-ROCを計算する
  - Precision、Recall、F1-Scoreを計算する
  - Top-K Accuracyを計算する
  - 100回分モデルとの比較を行う
  - 精度の変化を記録する
