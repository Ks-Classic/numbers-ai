# 01-03 精度評価と判断

## 概要
両データ範囲の精度を比較し、次ステップを決定する。目標値（AUC-ROC: 0.65以上）の達成を確認し、必要に応じて分割モデルを検討する。

## 前提条件

- Phase 1（ベースライン確立）が完了していること
- Phase 2（ハイパーパラメータチューニング）が完了していること

## 詳細タスク

### ステップ1: 精度評価と比較

- [ ] 両データ範囲の精度を比較する
  - 1000回分モデルのAUC-ROCを確認する
  - 4801回分モデルのAUC-ROCを確認する
  - その他のメトリクス（Precision、Recall、F1-Score）を比較する
  - データ範囲による精度変化を分析する

- [ ] AUC-ROC >= 0.65 の達成確認を行う
  - 目標達成の場合: ✅ 完了（Phase 4へ）
  - 目標未達の場合: Step 2へ

- [ ] 精度評価レポートを作成する
  - 評価結果をまとめる
  - グラフや表を作成する
  - ドキュメントに記録する

### ステップ2: パターン別精度分析（4801回分のみ・条件付き）

**条件**: AUC-ROC < 0.65 の場合のみ実施

- [ ] 統合モデルで各パターンの精度を分析する
  - A1パターンの精度を計算する
  - A2パターンの精度を計算する
  - B1パターンの精度を計算する
  - B2パターンの精度を計算する

- [ ] パターン間の精度差を確認する
  - 最大値 - 最小値を計算する
  - 最大値 - 最小値が0.1以上 → 分割モデルを検討（Step 3へ）
  - 最大値 - 最小値が0.1未満 → 分割モデルは不要

- [ ] パターン別精度分析レポートを作成する
  - 各パターンの精度をまとめる
  - パターン間の精度差を可視化する
  - ドキュメントに記録する

### ステップ3: 分割モデル（条件付き・4801回分のみ）

**条件**: AUC-ROC < 0.65 かつ パターン間の精度差が0.1以上

- [ ] 4801回分で4パターンごとに個別モデルを学習する
  - A1パターン専用モデルを学習する
  - A2パターン専用モデルを学習する
  - B1パターン専用モデルを学習する
  - B2パターン専用モデルを学習する

- [ ] 各パターンでハイパーパラメータチューニングを実施する
  - 各パターン専用モデルで最適パラメータを探索する
  - ハイパーパラメータチューニングの探索範囲を使用する
  - 各パターンの最適パラメータを記録する

- [ ] 統合モデル vs 分割モデルの比較を実施する
  - 精度比較を行う
  - 管理コスト比較を行う
  - 最終的なモデル選択を行う

- [ ] 分割モデル評価レポートを作成する
  - 統合モデルと分割モデルの比較結果をまとめる
  - 推奨モデルを決定する
  - ドキュメントに記録する

## 判断基準

### 目標達成の場合（AUC-ROC >= 0.65）

- ✅ Phase 4（最終モデル生成）へ進む
- 最適なモデルとパラメータで最終モデルを生成する

### 目標未達の場合（AUC-ROC < 0.65）

- パターン間の精度差が0.1以上: 分割モデルを検討する
- パターン間の精度差が0.1未満: 分割モデルは不要、他の改善策を検討する

## 注意事項

- 1000回分での分割モデルは検討しない（データ量が少ないため）
- 4801回分での分割モデルは条件付きで検討する
- 精度評価は複数のメトリクスで実施する（AUC-ROC、Precision、Recall、F1-Score）
- 評価結果は必ずドキュメントに記録する
