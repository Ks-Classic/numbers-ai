# 01-05 Weights & Biases導入とセットアップ

## 概要
Weights & Biases（W&B）を導入し、実験管理の基盤を構築する。ハイパーパラメータチューニングやアンサンブル学習の実験管理に使用する。

## 詳細タスク

- [ ] W&Bアカウントを作成する
  - W&Bの公式サイトにアクセスする
  - アカウントを作成する（無料プラン）
  - メールアドレスを確認する

- [ ] W&Bライブラリをインストールする
  - wandbをインストールする（pip install wandb）
  - バージョンを確認する
  - requirements.txtを更新する

- [ ] W&Bプロジェクトを作成する
  - W&Bダッシュボードでプロジェクトを作成する
  - プロジェクト名を設定する
  - プロジェクトの説明を記載する

- [ ] W&B初期化スクリプトを作成する
  - notebooks/03_wandb_setup.ipynbを作成する
  - wandb.init()を実行する
  - プロジェクト名とエンティティ名を設定する

- [ ] 学習スクリプトにW&B統合を追加する
  - notebooks/04_model_training.ipynbを更新する
  - wandb.configでハイパーパラメータを記録する
  - wandb.log()でメトリクスを記録する
  - wandb.log_model()でモデルを保存する（オプション）

- [ ] 初期ベースラインを記録する
  - 現在のモデルで学習を実行する
  - W&Bダッシュボードで結果を確認する
  - ベースラインとして記録する

- [ ] W&Bダッシュボードを確認する
  - 実験結果が正しく記録されていることを確認する
  - メトリクスの可視化を確認する
  - ハイパーパラメータの記録を確認する

## 注意事項

- W&Bは実験管理のためのオプションツール
- ハイパーパラメータチューニングでW&B Sweepsを使用することを推奨
- 無料プランでも基本的な機能は使用可能
