# 05-08 極CUBE基礎集計

## 概要
リハーサル数字非依存の全期間統計分析。極CUBEはリハーサル数字に依存しないため、初回1回目から最新回（6850回）までの全期間で一貫性のある分析が可能。サンプル数を最大化し、統計的信頼性を高める。

**極CUBEの仕様:**
- **N3のみ**（N4は対象外）
- **1パターンのみ**（A1/A2/B1/B2のどれでもない）
- **欠番補足なし**（0も含めて、すべて欠番補足しない、B1/B2と同じ）
- **中心0配置なし**（B1と同じ）
- **5行目の余りマスを0で埋める**（極CUBE固有）

## 前提条件

- [05-00_極CUBEルール定義と生成スクリプト作成.md](./05-00_極CUBEルール定義と生成スクリプト作成.md)が完了していること
- 極CUBE生成スクリプト（`scripts/generate_extreme_cube.py`）が利用可能であること
- **注意**: 極CUBEはオンデマンド生成（`generate_extreme_cube()`を使用）または既存のJSONファイルから読み込む
- ファイル保存は分析用途のみ（`data/extreme_cubes/`ディレクトリ、`.gitignore`に追加）
- 全期間（1-6850回）の極CUBEを生成する場合は、分析用途として個別JSONファイル形式で保存可能

## 詳細タスク

### Phase 1: 全期間基礎集計（推奨）

- [ ] 極CUBE内での当選番号出現分析を実施する
  - 各回号について、極CUBE内での当選番号出現を分析する（N3のみ、1パターンのみ）
  - つながり判定（縦横斜め）を実施する
  - 出現位置の詳細分析を実施する
  - 旧CUBEとの比較を実施する

- [ ] 数字の出現パターンを集計する
  - 数字(0-9) × 桁位置ごとに出現頻度を集計する（全期間、N3のみ）
  - 各桁位置での数字分布を集計する
  - よく出現する数字の組み合わせを抽出する

- [ ] 時系列・周期性分析を実施する
  - 曜日分析を実施する
  - 月相（月齢）分析を実施する
  - 季節・月分析を実施する
  - 年・四半期分析を実施する

- [ ] 全期間基礎集計の結果を出力する
  - CSV形式で出力する（`data/extreme_cube_full_period_analysis.csv`）
  - JSON形式で出力する（`data/extreme_cube_full_period_analysis.json`）

### Phase 2: 並び型分析（2,3,4行目）

- [ ] 並び型の定義と分類
  - V字型、逆V字型、L字型、逆L字型、┌字型、└字型、┘字型、┐字型、横一文字型、縦一文字型、斜め型（右下/左下/右上/左上）などの定義
  - 各型の全パターン列挙（V字型は12パターン確定）
  - つながりの判定ロジック（横・斜め・縦）

- [ ] 並び型判定スクリプトの作成
  - 極CUBEの2,3,4行目で当選数字の出現位置を取得
  - 各回号について型を判定
  - 判定結果をCSV/JSON形式で出力

- [ ] 曜日ごとの型の出現傾向分析
  - 過去150回、1000回、2000回、全期間（約6800回）での分析
  - 各曜日（月/火/水/木/金）ごとの型の出現頻度を集計
  - 統計的有意性の検証（カイ二乗検定など）
  - サンプル数拡張時の傾向の持続性を検証

- [ ] 曜日以外の要素での傾向分析
  - 月（1-12月）ごとの型の出現傾向
  - 月の第何週目（1-5週目）ごとの型の出現傾向
  - 回数の区切り（1000回ごと、500回ごとなど）での型の出現傾向
  - 年・四半期ごとの型の出現傾向
  - 各要素での統計的有意性を検証

- [ ] 階層的分析の実施
  - Phase 1: 基本型（8-10種類）で全体像を把握
  - Phase 2: 有意な型のみを細分化（例：横一文字型 → 2行目/3行目/4行目）
  - 統計的検出力を維持しながら、実用的な予測ルールを導出

- [ ] 分析結果の出力
  - CSV形式で出力（`data/extreme_cube_pattern_analysis.csv`）
  - JSON形式で出力（`data/extreme_cube_pattern_analysis.json`）
  - 統計的有意性の検証結果を含める

### Phase 3: 区切り別比較分析（検証用）

- [ ] 週5実施開始前後比較を実施する
  - 区切り: 1340回前後（2004年7月1日頃）
  - 当選番号の数字分布を比較する
  - CUBE内出現率を比較する
  - 周期性の変化を分析する

- [ ] リハーサル数字導入前後比較を実施する
  - 区切り: 4801回前後（2024年1月18日頃）
  - 当選番号の数字分布を比較する
  - CUBE内出現率を比較する
  - 周期性の変化を分析する

- [ ] 最新5年間分析を実施する
  - 区切り: 約5000回以降
  - 最新トレンドを把握する
  - 最近の傾向変化を分析する

- [ ] 区切り別比較分析の結果を出力する
  - CSV形式で出力する（`data/extreme_cube_segmented_comparison.csv`）
  - JSON形式で出力する（`data/extreme_cube_segmented_comparison.json`）
