# 05-03 時系列・周期性分析

## 概要
時間軸に基づく周期性とトレンドの検出。曜日、月相（月齢）、季節、月、年・四半期などの時間軸で当選番号の出現パターンを分析し、周期性やトレンドを特定する。

## 詳細タスク

### Phase 1: データ準備と時間軸の計算

- [ ] データ読み込みスクリプトを作成する
  - `scripts/analyze_temporal_patterns.py`を作成する
  - `data/past_results.csv`を読み込む
  - 4801-6850回のデータをフィルタリングする
  - 抽選日（draw_date）から時間軸情報を抽出する

- [ ] 曜日を計算する
  - 各回号の抽選日から曜日（月/火/水/木/金）を計算する
  - 曜日情報をデータに追加する

- [ ] 月相（月齢）を計算する
  - 各回号の抽選日から月齢（0-29日）を計算する
  - 月齢計算ライブラリ（例: `astral`、`skyfield`）を使用する
  - 新月（0-2日）、上弦（7-9日）、満月（14-16日）、下弦（21-23日）を判定する
  - 月齢情報をデータに追加する

- [ ] 季節・月を計算する
  - 各回号の抽選日から季節（春/夏/秋/冬）を計算する
  - 各回号の抽選日から月（1-12月）を抽出する
  - 季節・月情報をデータに追加する

- [ ] 年・四半期を計算する
  - 各回号の抽選日から年を抽出する
  - 各回号の抽選日から四半期（Q1-Q4）を計算する
  - 年・四半期情報をデータに追加する

### Phase 2: 曜日分析

- [ ] 曜日別の基本統計を集計する
  - 各曜日（月/火/水/木/金） × 対象(N3/N4)ごとに抽選回数を集計する
  - 各曜日の当選番号の数字分布（0-9の出現頻度）を集計する

- [ ] 曜日別のリハーサル数字と当選番号の重なり率を計算する
  - 各曜日 × 対象(N3/N4)ごとに、リハーサル数字と当選番号の重なり率を計算する
  - 重なり率 = 重なりがあった回数 / 全回数 × 100

- [ ] 曜日別のCUBE内出現率を計算する
  - 各曜日 × 対象(N3/N4)ごとに、CUBE内出現率を計算する
  - CUBE内出現率 = CUBE内に出現した回数 / 全回数 × 100

- [ ] 曜日別の傾向を分析する
  - 特定の数字が出やすい曜日を特定する
  - 曜日別の数字分布の違いを統計的に検定する（カイ二乗検定など）

- [ ] 曜日分析の結果を出力する
  - CSV形式で出力する（`data/weekday_analysis.csv`）
  - JSON形式で出力する（`data/weekday_analysis.json`）

### Phase 3: 月相（月齢）分析

- [ ] 月齢別の基本統計を集計する
  - 各月齢（0-29日） × 対象(N3/N4)ごとに抽選回数を集計する
  - 月齢別の当選番号の数字分布を集計する

- [ ] 月相フェーズ別の傾向を分析する
  - 新月（0-2日）、上弦（7-9日）、満月（14-16日）、下弦（21-23日）での傾向を分析する
  - 各月相フェーズでの当選番号の数字分布を集計する

- [ ] 月齢別のリハーサル数字と当選番号の重なり率を計算する
  - 各月齢 × 対象(N3/N4)ごとに、リハーサル数字と当選番号の重なり率を計算する

- [ ] 月齢別のCUBE内出現率を計算する
  - 各月齢 × 対象(N3/N4)ごとに、CUBE内出現率を計算する

- [ ] 月相分析の結果を出力する
  - CSV形式で出力する（`data/moon_phase_analysis.csv`）
  - JSON形式で出力する（`data/moon_phase_analysis.json`）

### Phase 4: 季節・月分析

- [ ] 季節別の基本統計を集計する
  - 各季節（春/夏/秋/冬） × 対象(N3/N4)ごとに抽選回数を集計する
  - 季節別の当選番号の数字分布を集計する

- [ ] 月別の基本統計を集計する
  - 各月（1-12月） × 対象(N3/N4)ごとに抽選回数を集計する
  - 月別の当選番号の数字分布を集計する

- [ ] 季節別・月別のリハーサル数字と当選番号の重なり率を計算する
  - 各季節 × 対象(N3/N4)ごとに、リハーサル数字と当選番号の重なり率を計算する
  - 各月 × 対象(N3/N4)ごとに、リハーサル数字と当選番号の重なり率を計算する

- [ ] 季節別・月別のCUBE内出現率を計算する
  - 各季節 × 対象(N3/N4)ごとに、CUBE内出現率を計算する
  - 各月 × 対象(N3/N4)ごとに、CUBE内出現率を計算する

- [ ] 季節・月分析の結果を出力する
  - CSV形式で出力する（`data/season_month_analysis.csv`）
  - JSON形式で出力する（`data/season_month_analysis.json`）

### Phase 5: 年・四半期分析

- [ ] 年次トレンドを分析する
