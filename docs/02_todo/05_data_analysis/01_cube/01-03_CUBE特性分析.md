# 05-05 CUBE特性分析

## 概要
CUBEの構造特性と当選番号との関係性。CUBEのサイズ、密度、メイン行の特性を分析し、CUBE内での当選番号の出現位置や傾向を把握する。

## 詳細タスク

### Phase 1: CUBE生成と構造特性の抽出

- [ ] CUBE生成スクリプトを作成する
  - `scripts/analyze_cube_characteristics.py`を作成する
  - 旧keisenと新keisenの両方でCUBEを生成する
  - 各回号 × パターン × 対象 × keisen種別でCUBEを生成する
  - **注意**: CUBEはオンデマンド生成（`chart_generator.generate_chart()`を使用）または既存のJSONファイルから読み込む
  - ファイル保存は分析用途のみ（`data/analysis/`ディレクトリ、`.gitignore`に追加）

- [ ] CUBEの構造特性を抽出する
  - CUBEの行数を記録する
  - CUBEの列数（固定8列）を記録する
  - CUBE内の数字の総数をカウントする
  - CUBE内の空マスの数をカウントする
  - CUBEの密度を計算する（数字数 / 総マス数 × 100）

- [ ] メイン行の特性を抽出する
  - メイン行の数を記録する
  - 各メイン行の数字数を記録する
  - メイン行の数字の種類を記録する

- [ ] CUBE構造特性の結果を出力する
  - CSV形式で出力する（`data/cube_structure_characteristics.csv`）
  - JSON形式で出力する（`data/cube_structure_characteristics.json`）

### Phase 2: CUBEと当選番号の関係分析

- [ ] 当選番号がCUBE内に出現したかを判定する
  - 各回号 × パターン × 対象 × keisen種別について、当選番号がCUBE内に出現したかを判定する
  - 出現回数をカウントする
  - 出現率を計算する（出現回数 / 全回数 × 100）

- [ ] 出現パターン別の出現率を集計する
  - 縦方向のつながりでの出現率を計算する
  - 横方向のつながりでの出現率を計算する
  - 斜め方向のつながりでの出現率を計算する
  - 複合パターンでの出現率を計算する

- [ ] 当選番号が出現した位置の分布を集計する
  - 行・列ごとの出現回数を集計する
  - 出現位置の分布を可視化する（ヒートマップなど）

- [ ] 当選番号が出現した位置の傾向を分析する
  - 特定の行・列に集中するかを統計的に検定する（カイ二乗検定など）
  - 出現位置の傾向を可視化する

- [ ] CUBEと当選番号の関係分析の結果を出力する
  - CSV形式で出力する（`data/cube_winning_relationship_analysis.csv`）
  - JSON形式で出力する（`data/cube_winning_relationship_analysis.json`）

### Phase 3: CUBE特性と当選番号出現率の相関分析

- [ ] CUBE特性と当選番号出現率の相関を計算する
  - CUBEの行数と当選番号出現率の相関を計算する
  - CUBEの密度と当選番号出現率の相関を計算する
  - メイン行の数と当選番号出現率の相関を計算する

- [ ] 相関分析の結果を出力する
  - CSV形式で出力する（`data/cube_characteristics_correlation.csv`）
  - JSON形式で出力する（`data/cube_characteristics_correlation.json`）

### Phase 4: CUBE特性分析レポート作成

- [ ] 分析結果のサマリーを作成する
  - CUBE構造特性のサマリーを作成する
  - CUBEと当選番号の関係のサマリーを作成する
  - 相関分析のサマリーを作成する

- [ ] 可視化を作成する
  - CUBE構造特性の分布グラフを作成する（行数、密度など）
  - 当選番号出現率の比較グラフを作成する（旧keisen vs 新keisen）
  - 出現位置の分布ヒートマップを作成する
  - 相関分析の散布図を作成する

- [ ] レポートを出力する
  - Markdown形式でレポートを作成する（`docs/report/cube_characteristics_analysis_report.md`）
  - 分析結果のサマリー、可視化、統計情報を含める
