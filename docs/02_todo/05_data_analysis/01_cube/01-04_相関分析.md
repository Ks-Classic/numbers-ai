# 05-06 相関分析

## 概要
変数間の相関関係と相互依存性の分析。リハーサル数字と当選番号、N3とN4、前回・前々回との相関を分析し、変数間の関係性を把握する。

## 詳細タスク

### Phase 1: リハーサル数字と当選番号の相関分析

- [ ] データ読み込みスクリプトを作成する
  - `scripts/analyze_correlations.py`を作成する
  - `data/past_results.csv`を読み込む
  - 4801-6850回のデータをフィルタリングする
  - リハーサル数字と当選番号を抽出する

- [ ] リハーサル数字と当選番号の相関係数を計算する
  - 対象(N3/N4)ごとに、各桁ごとの相関係数を計算する
  - ピアソンの相関係数を使用する
  - 統計的有意性を検定する（p値）

- [ ] リハーサル数字と当選番号の距離の分布を集計する
  - 各回号について、リハーサル数字と当選番号の距離を計算する
  - 距離の分布（平均、中央値、標準偏差など）を集計する
  - 距離の分布を可視化する（ヒストグラムなど）

- [ ] リハーサル数字と当選番号の重なりの分布を集計する
  - 各回号について、リハーサル数字と当選番号の重なりを計算する
  - 重なりの分布（平均、中央値、標準偏差など）を集計する
  - 重なりの分布を可視化する（ヒストグラムなど）

- [ ] リハーサル数字と当選番号の相関分析の結果を出力する
  - CSV形式で出力する（`data/rehearsal_winning_correlation.csv`）
  - JSON形式で出力する（`data/rehearsal_winning_correlation.json`）
