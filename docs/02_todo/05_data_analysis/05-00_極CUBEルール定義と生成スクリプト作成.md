# 05-00 極CUBEルール定義と生成スクリプト作成

## 概要
極CUBEの生成ルールを定義し、自動生成スクリプトを作成する。極CUBEはリハーサル数字に依存せず、全期間（1-6850回）で一貫性のある分析が可能なCUBE生成方法を確立する。

**極CUBEの仕様:**
- **N3のみ**（N4は対象外）
- **1パターンのみ**（A1/A2/B1/B2のどれでもない）
- **欠番補足なし**（0も含めて、すべて欠番補足しない、B1/B2と同じ）
- **中心0配置なし**（B1と同じ）
- **5行目の余りマスを0で埋める**（極CUBE固有）

## 注意事項

- 極CUBE生成ルールは既存のCUBE生成ルールとは異なるため、慎重に定義する必要がある
- 全期間（1-6850回）でCUBEを生成するため、処理時間とメモリ使用量に注意する
- 処理時間: 約1〜2時間（全期間データによる、N3のみ、1パターンのみのため）
- メモリ使用量: 約1〜2GB（N3のみ、1パターンのみのため）
- 生成されたCUBEは`data/extreme_cubes/n3/`ディレクトリに保存する（パターンサブディレクトリ不要）
- ドキュメントは`docs/01_design/CUBE生成ルール.md`に保存する
