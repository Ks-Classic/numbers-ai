# 05-04 数値パターン分析

## 概要
数字の出現パターンと組み合わせ特性の分析。各数字（0-9）の出現頻度、よく出現する数字の組み合わせ、連続性・周期性を分析し、数字の出現傾向を把握する。

## 詳細タスク

### Phase 1: 数字の出現パターン分析

- [ ] データ読み込みスクリプトを作成する
  - `scripts/analyze_digit_patterns.py`を作成する
  - `data/past_results.csv`を読み込む
  - 4801-6850回のデータをフィルタリングする
  - N3とN4の当選番号を抽出する

- [ ] 各数字の出現回数（全体）を集計する
  - 数字(0-9) × 対象(N3/N4)ごとに出現回数をカウントする
  - 各数字の出現率（全体）を計算する（出現回数 / 全回数 × 100）

- [ ] 各桁位置での各数字の出現回数を集計する
  - 数字(0-9) × 対象(N3/N4) × 桁位置ごとに出現回数をカウントする
  - N3: 百の位/十の位/一の位
  - N4: 千の位/百の位/十の位/一の位
  - 各桁位置での各数字の出現率を計算する

- [ ] 数字の出現パターン分析の結果を出力する
  - CSV形式で出力する（`data/digit_pattern_analysis.csv`）
  - JSON形式で出力する（`data/digit_pattern_analysis.json`）

### Phase 2: 数字の組み合わせ分析

- [ ] よく出現する数字の組み合わせを抽出する
  - N3: 3桁の組み合わせを抽出する
  - N4: 4桁の組み合わせを抽出する
  - 各組み合わせの出現回数をカウントする

- [ ] 組み合わせの出現率を計算する
  - 各組み合わせの出現率を計算する（出現回数 / 全回数 × 100）
  - 出現率の高い組み合わせを上位N件抽出する（例: 上位100件）

- [ ] 組み合わせの特徴を分析する
  - よく出現する数字の組み合わせの特徴を分析する（同じ数字の重複、連続性など）
  - 組み合わせの出現パターンを可視化する

- [ ] 数字の組み合わせ分析の結果を出力する
  - CSV形式で出力する（`data/digit_combination_analysis.csv`）
  - JSON形式で出力する（`data/digit_combination_analysis.json`）

### Phase 3: 連続性・周期性分析

- [ ] 同じ数字が連続して出現した回数を集計する
  - 対象(N3/N4)ごとに、同じ数字が連続して出現した回数をカウントする
  - 連続する数字の種類を記録する

- [ ] 同じ数字が連続する確率を計算する
  - 連続出現回数 / 全回数 × 100で確率を計算する
  - 連続する数字の種類ごとの確率を計算する

- [ ] 特定の数字が周期的に出現するパターンを検出する
  - 各数字(0-9)について、周期的な出現パターンを検出する
  - 周期の長さ（何回おきに出現するか）を計算する
  - 自己相関関数を使用して周期性を検出する

- [ ] 周期の信頼性を評価する
  - 統計的有意性を検定する（例: コルモゴロフ-スミルノフ検定）
  - 周期の信頼性スコアを計算する

- [ ] 連続性・周期性分析の結果を出力する
  - CSV形式で出力する（`data/digit_continuity_periodicity_analysis.csv`）
  - JSON形式で出力する（`data/digit_continuity_periodicity_analysis.json`）

### Phase 4: 数値パターン分析レポート作成

- [ ] 分析結果のサマリーを作成する
  - 数字の出現パターンのサマリーを作成する
  - 数字の組み合わせのサマリーを作成する
  - 連続性・周期性のサマリーを作成する

- [ ] 可視化を作成する
  - 各数字の出現頻度の棒グラフを作成する
  - 各桁位置での数字分布のヒートマップを作成する
  - よく出現する組み合わせのランキンググラフを作成する
  - 周期性の可視化を作成する（時系列グラフ、自己相関グラフなど）

- [ ] レポートを出力する
  - Markdown形式でレポートを作成する（`docs/report/digit_pattern_analysis_report.md`）
  - 分析結果のサマリー、可視化、統計情報を含める
