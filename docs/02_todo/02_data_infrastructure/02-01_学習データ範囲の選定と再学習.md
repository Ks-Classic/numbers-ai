# 02-01 学習データ範囲の選定と再学習

## 概要
学習データ範囲を選定し、AIモデルを再学習する。現在の精度（AUC-ROC: 0.51）を向上させるため、段階的アプローチで改善を実施する。

## 詳細タスク

- [ ] 学習データ範囲の選択肢を明確化する
  - 1000回分（5849回〜6849回）の選択肢を検討する
  - 4801回から最新回までの選択肢を検討する
  - 各選択肢のメリット・デメリットを比較する
  - 最適な選択肢を決定する

- [ ] 学習設定ファイル（config.py）を作成する
  - 学習範囲を設定ファイルで管理可能にする
  - TRAIN_SIZE = 1000に設定する（選択肢1）
  - MIN_ROUND = 4801に対応する（選択肢2）
  - 基準回号を最新回号に自動取得するように変更する

- [ ] データセットを準備する
  - notebooks/01_data_preparation.ipynbを更新する
  - 1,000回分のデータを読み込む処理を実装する
  - 4801回から最新回までのデータ読み込みに対応する
  - 設定ファイル（config.py）から設定を読み込むように変更する
  - 基準回号を最新回号（第6849回）に更新する

- [ ] Notebookファイルを設定ファイル対応に更新する
  - notebooks/01_data_preparation.ipynb: config.pyを使用、MIN_ROUND対応
  - notebooks/03_feature_engineering.ipynb: config.pyを使用
  - notebooks/04_model_training.ipynb: config.pyを使用

- [ ] データクリーニングの改善を実施する
  - 先頭0が失われる問題を修正する（zfill処理を追加）
  - データクリーニング後の行数を確認する
  - データ品質を向上させる

- [ ] 学習データの生成を実行する
  - 1000回分のデータが正常に生成されることを確認する（5849回〜6849回）
