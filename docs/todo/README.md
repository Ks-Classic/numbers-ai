## 📝 重要な方針決定

### AI精度改善について
- **特徴量エンジニアリング実験はスキップ**: 全特徴量を使用し、ハイパーパラメータチューニングに集中する
- **1000回分での分割モデルは検討しない**: データ量が少ないため、統合モデルのみ
- **4801回分での分割モデルは条件付き**: 統合モデル + ハイパーパラメータチューニング後、精度が目標未達かつパターン間の精度差が大きい場合のみ検討

### マイクロサービス化について
- **現時点では実施不要**: Phase 4で実施予定
- **パターンB（0あり）実装**: ✅ 完了済み（4パターン対応）
- **統合モデル採用**: 分割モデルを別サービスにする必要はない
