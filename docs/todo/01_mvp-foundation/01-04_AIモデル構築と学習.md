# 01-04 AIモデル構築と学習

## 概要
XGBoostを使用した統合モデルを構築し、過去データで学習させる（4パターン統合、6モデル構成）。

**統合モデルの特徴:**
- 4パターン（A1/A2/B1/B2）すべてのデータで学習
- パターンIDを特徴量として追加（A1=0, A2=1, B1=2, B2=3）
- データ量が4倍になる（100回分 → 400回分相当）
- パターン間の共通パターンと相違点を同時に学習

**モデル構成:**
- 軸数字予測: N3/N4のみで分ける（2モデル）※ボックス/ストレートで分けない理由: ラベルが「その数字が当選番号に含まれたか」であり、順序は関係ない
- 組み合わせ予測: N3/N4 × ボックス/ストレートで分ける（4モデル）※順序が重要

## 詳細タスク

- [ ] Jupyter Notebook環境をセットアップする
  - notebooks/ディレクトリを作成する
  - 必要なPythonライブラリをインストールする（pandas, numpy, xgboost, scikit-learn等）
  - requirements.txtを作成する

- [ ] データ前処理スクリプトを作成する
  - notebooks/01_data_preparation.ipynbを作成する
  - CSVデータを読み込む処理を実装する
  - データクリーニングを実装する（欠損値、異常値の処理）
  - 学習用データセットを準備する（直近100回分）

- [ ] 特徴量エンジニアリングの実装を開始する
  - notebooks/02_chart_generation.ipynbを作成する
  - 各回号に対して4パターン（A1/A2/B1/B2）の予測表を生成する処理を実装する
  - 予測表から特徴量を抽出する処理を実装する（基本的な特徴量のみ）
  - パターンID（A1=0, A2=1, B1=2, B2=3）を特徴量として追加する処理を実装する

- [ ] 特徴量計算関数を実装する
  - 形状特徴の計算関数を実装する（線の長さ、曲がりの回数等）
  - 位置特徴の計算関数を実装する（重心X/Y座標、端からの距離等）
  - 関係性特徴の計算関数を実装する（リハーサルとの距離、重なり度等）
  - 集約特徴の計算関数を実装する（出現頻度、分散度等）

- [ ] 軸数字予測モデルの学習データを生成する
  - 各回号×4パターン×各数字（0-9）のサンプルを生成する
  - パターンIDを特徴量として追加する（4次元のone-hot encoding）
  - 特徴量ベクトルを作成する
  - ラベル（0または1）を生成する（当選番号に含まれたか）
  - データセットを分割する（train/val）
  - データ量: 100回分 × 4パターン × 10数字 = 4,000サンプル

- [ ] 組み合わせ予測モデルの学習データを生成する
  - 各回号×4パターン×全組み合わせ（またはサンプリング）のサンプルを生成する
  - パターンIDを特徴量として追加する（4次元のone-hot encoding）
  - 特徴量ベクトルを作成する
  - ラベル（0または1）を生成する（当選したか）
  - データセットを分割する（train/val）
  - データ量: 100回分 × 4パターン × 数百〜数千サンプル = 数万〜数十万サンプル

- [ ] XGBoostモデルの学習スクリプトを作成する
  - notebooks/04_model_training.ipynbを作成する
  - ハイパーパラメータを設定する（max_depth=6, learning_rate=0.05等）
  - モデルを学習させる処理を実装する
  - Early stoppingを設定する

- [ ] 6つの統合モデルを学習する
  - 各モデルは4パターンすべてのデータで学習する
  - 軸数字予測モデル（2モデル）:
    - N3軸数字予測モデルを学習する（n3_axis.pkl）
    - N4軸数字予測モデルを学習する（n4_axis.pkl）
    - 注意: ボックス/ストレートで分けない（順序は関係ないため）
  - 組み合わせ予測モデル（4モデル）:
    - N3ボックス組み合わせ予測モデルを学習する（n3_box_comb.pkl）
    - N3ストレート組み合わせ予測モデルを学習する（n3_straight_comb.pkl）
    - N4ボックス組み合わせ予測モデルを学習する（n4_box_comb.pkl）
    - N4ストレート組み合わせ予測モデルを学習する（n4_straight_comb.pkl）

- [ ] モデル評価を実施する
  - AUC-ROCを計算する
  - Precision、Recall、F1-Scoreを計算する
  - Top-K Accuracyを計算する
  - 評価結果を記録する

- [ ] 学習済みモデルを保存する
  - data/models/ディレクトリを作成する
  - 各モデルを.pkl形式で保存する
  - モデルファイル名を統一する（n3_axis.pkl, n3_box_comb.pkl, n3_straight_comb.pkl, n4_axis.pkl, n4_box_comb.pkl, n4_straight_comb.pkl）

- [ ] モデル読み込みユーティリティを作成する
  - Python側でモデルを読み込む関数を実装する
  - モデルファイルの存在確認を実装する
  - エラーハンドリングを実装する
