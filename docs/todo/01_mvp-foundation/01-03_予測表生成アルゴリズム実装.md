# 01-03 予測表生成アルゴリズム実装

## 概要
中国罫線理論に基づく予測表生成アルゴリズムを実装する（パターンA: 0なし）。

## 詳細タスク

- [ ] 予測表生成の型定義を作成する
  - src/types/chart.tsを作成する
  - ChartGrid型を定義する（(number | null)[][]）
  - ChartData型を定義する（grid, rows, cols, pattern, target, sourceDigits）
  - MainRow型を定義する（elements, rowIndex）

- [ ] 予測出目抽出ロジックを実装する
  - 前回（round_number-1）の当選番号を取得する関数を実装する
  - 前々回（round_number-2）の当選番号を取得する関数を実装する
  - keisen_master.jsonから各桁の予測出目を取得する関数を実装する
  - 全桁の予測出目を結合してsource_listを生成する関数を実装する

- [ ] パターンAの元数字リスト作成ロジックを実装する
  - source_listに0-9の欠番をすべて追加する関数を実装する
  - 昇順ソートを行う関数を実装する
  - 重複削除の処理を実装する

- [ ] メイン行組み立てロジックを実装する
  - buildMainRows関数を作成する
  - uniqueDigitsを抽出する処理を実装する
  - uniqueDigits.length >= 4の場合の処理を実装する（最初の4要素を選択）
  - uniqueDigits.length < 4の場合の処理を実装する（最大値を繰り返し追加）
  - numsから使用した数字を削除する処理を実装する
  - ループ終了条件を実装する（numsが空になるまで）

- [ ] グリッド初期配置ロジックを実装する
  - initializeGrid関数を作成する
  - rows = mainRows.length * 2で行数を計算する
  - cols = 8で列数を設定する
  - 2次元配列を初期化する（初期値null）
  - mainRowsの各要素をグリッドに配置する（行i*2、列j*2）

- [ ] 裏数字ルール（縦パス）を実装する
  - inverse関数を実装する（(n + 5) % 10）
  - applyVerticalInverse関数を作成する
  - 上から下へ順に処理するループを実装する
  - nullかつ上に値がある場合に裏数字を配置する処理を実装する
  - 更新がなくなるまで繰り返す処理を実装する

- [ ] 裏数字ルール（横パス）を実装する
  - applyHorizontalInverse関数を作成する
  - 左から右へ順に処理するループを実装する
  - nullかつ左に値がある場合に裏数字を配置する処理を実装する
  - 更新がなくなるまで繰り返す処理を実装する

- [ ] 余りマスルールを実装する
  - applyRemainingCopy関数を作成する
  - nullかつ上に値がある場合に上から値をコピーする処理を実装する
  - 更新がなくなるまで繰り返す処理を実装する

- [ ] 予測表生成の統合関数を実装する
  - src/lib/chart-generator/index.tsにgenerateChart関数を作成する
  - 全ステップを順番に実行する処理を実装する
  - ChartDataオブジェクトを返す処理を実装する

- [ ] ユニットテストを実装する
  - 予測表生成の正常系テストを書く
  - 各ステップのユニットテストを書く
  - エッジケースのテストを書く（データ不足、境界値等）
  - テストを実行して確認する

- [ ] 予測表生成の結果を検証する
  - 生成されたグリッドが8列であることを確認する
  - 行数がmainRows.length * 2であることを確認する
  - nullが残っていないことを確認する
  - 実際のデータで動作確認をする
