# 02-04 1,800回分データでの再学習

## 概要
学習データを1,800回分に拡大し、AIモデルを再学習する。

## 詳細タスク

- [ ] Supabaseから1,800回分のデータを取得する
  - データ取得スクリプトを作成する
  - リハーサル数字が存在するデータをフィルタリングする
  - データの整合性を確認する

- [ ] データセットを準備する
  - notebooks/01_data_preparation.ipynbを更新する
  - 1,800回分のデータを読み込む処理を実装する
  - データクリーニングを実施する

- [ ] 予測表生成処理を並列化する（オプション）
  - 大量データの処理時間を短縮する
  - マルチプロセッシングまたは並列処理を実装する
  - 処理時間を測定する

- [ ] 特徴量計算処理を最適化する
  - 処理速度を改善する
  - 不要な計算を削減する
  - キャッシュを活用する

- [ ] 学習データセットを生成する
  - 軸数字予測モデル用のデータセットを生成する
  - 組み合わせ予測モデル用のデータセットを生成する
  - データセットのサイズを確認する

- [ ] モデルを再学習する
  - notebooks/04_model_training.ipynbを更新する
  - 8つのモデルを順番に学習する
  - 学習時間を記録する

- [ ] モデル評価を実施する
  - AUC-ROCを計算する
  - Precision、Recall、F1-Scoreを計算する
  - Top-K Accuracyを計算する
  - 100回分データとの精度を比較する

- [ ] モデルを保存する
  - 新しいモデルファイルを保存する
  - モデルファイル名にバージョンを付与する
  - 古いモデルファイルをバックアップする

- [ ] モデルをデプロイする
  - 新しいモデルファイルを本番環境に配置する
  - APIが新しいモデルを読み込むことを確認する
  - 動作確認をする
