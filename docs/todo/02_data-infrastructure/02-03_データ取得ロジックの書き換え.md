# 02-03 データ取得ロジックの書き換え

## 概要
CSV読み込み処理をSupabaseクエリに置き換え、データ取得ロジックを更新する。

## 詳細タスク

- [ ] Supabaseクエリ関数を作成する
  - src/lib/data-loader/supabase-past-results.tsを作成する
  - 回号でデータを取得する関数を実装する
  - 日付範囲でデータを取得する関数を実装する
  - 最新N件を取得する関数を実装する

- [ ] 過去実績データ取得処理を更新する
  - src/lib/data-loader/past-results.tsを更新する
  - CSV読み込み処理をSupabaseクエリに置き換える
  - 型定義を更新する
  - エラーハンドリングを実装する

- [ ] 予測表生成処理を更新する
  - 前回・前々回のデータ取得処理をSupabaseクエリに置き換える
  - データが存在しない場合の処理を実装する
  - パフォーマンスを最適化する

- [ ] キャッシュ処理を実装する（オプション）

- [ ] パフォーマンステストを実施する

- [ ] エラーハンドリングを強化する

- [ ] データ更新処理を実装する（将来用）

  - データ更新APIを実装する（管理者用）
