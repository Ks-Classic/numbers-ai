# 特徴量エンジニアリング改善タスク

**作成日**: 2025年1月  
**ステータス**: 🔄 進行中

## 目的

リハーサル数字関連特徴量の強化と形状・位置特徴量の追加により、モデルの予測性能を向上させる。

**目標**: AUC-ROC 0.55-0.60

## 実施内容

### ✅ 第1段階: リハーサル数字関連特徴量の強化（完了）

#### 追加された特徴量（9次元）

1. **距離帯別特徴量（4次元）**:
   - `rehearsal_distance_bin_0_1`: 距離0-1の割合
   - `rehearsal_distance_bin_1_2`: 距離1-2の割合
   - `rehearsal_distance_bin_2_3`: 距離2-3の割合
   - `rehearsal_distance_bin_3_plus`: 距離3以上の割合

2. **角度特徴量（1次元）**:
   - `rehearsal_angle`: 候補数字とリハーサル数字の重心を結ぶ線の角度（ラジアン、0-π）

3. **桁ごとの距離統計特徴量（4次元）**:
   - `rehearsal_digit_0_mean_distance`: リハーサル数字の1桁目との平均距離
   - `rehearsal_digit_1_mean_distance`: リハーサル数字の2桁目との平均距離
   - `rehearsal_digit_2_mean_distance`: リハーサル数字の3桁目との平均距離
   - `rehearsal_digit_3_mean_distance`: リハーサル数字の4桁目との平均距離（N4の場合）

### ✅ 第2段階: 形状・位置特徴量の追加（完了）

#### 追加された形状特徴量（3次元）

- `diagonal_line_length`: 対角線方向の連続長（左上→右下、右上→左下の両方向をチェック）
- `clustering_coefficient`: クラスタリング係数（0-1、1が最もクラスタリングされている）
- `shape_complexity`: 形状の複雑度（0-1、曲がり回数と分散度を組み合わせ）

#### 追加された位置特徴量（5次元）

- `quadrant_top_left`: 左上象限の点の割合
- `quadrant_top_right`: 右上象限の点の割合
- `quadrant_bottom_left`: 左下象限の点の割合
- `quadrant_bottom_right`: 右下象限の点の割合
- `edge_proximity`: エッジ近接度（0-1、1が最もエッジに近い）
