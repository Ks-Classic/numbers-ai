# 05-01 共通UIコンポーネント実装

## 概要
shadcn/uiを使用した共通UIコンポーネントを実装し、デザインシステムを構築する。

## 詳細タスク

- [ ] 必要なshadcn/uiコンポーネントをインストールする
  - accordionコンポーネントをインストールする
  - modalコンポーネントをインストールする
  - その他、必要なコンポーネントをインストールする

- [ ] カラーパレットを設定する
  - tailwind.config.tsでカラーパレットを定義する
  - プライマリカラーを設定する
  - セマンティックカラーを設定する

- [ ] タイポグラフィを設定する
  - フォントファミリーを設定する（Inter, Noto Sans JP）
  - フォントサイズを定義する
  - 行間を設定する

- [ ] 共通スタイルを定義する
  - globals.cssに共通スタイルを追加する
  - カスタムユーティリティクラスを定義する
  - アニメーションを定義する

- [ ] レイアウトコンポーネントを実装する
  - Headerコンポーネントを実装する
  - Navigationコンポーネントを実装する
  - Footerコンポーネントを実装する（必要に応じて）

- [ ] フォームコンポーネントを実装する
  - RoundInputコンポーネントを実装する
  - RehearsalInputコンポーネントを実装する
  - バリデーション機能を実装する

- [ ] カードコンポーネントを実装する
  - 予測結果表示用のカードコンポーネントを実装する
  - 軸数字候補用のカードコンポーネントを実装する
  - スタイリングを統一する

- [ ] ローディングコンポーネントを実装する
  - スピナーコンポーネントを実装する
  - プログレスバーコンポーネントを実装する
  - ローディングメッセージ表示を実装する

- [ ] エラー表示コンポーネントを実装する
  - エラーメッセージ表示コンポーネントを実装する
  - エラーアイコンを追加する
  - スタイリングを統一する

- [ ] モーダルコンポーネントを実装する
  - スコア説明モーダルを実装する
  - 確認ダイアログモーダルを実装する
  - スタイリングを統一する

- [ ] アコーディオンコンポーネントを実装する
  - 軸数字候補展開用のアコーディオンを実装する
  - スムーズなアニメーションを実装する
  - アクセシビリティを確保する

- [ ] コンポーネントのドキュメントを作成する
  - Storybookを導入する（オプション）
  - コンポーネントの使い方をドキュメント化する
  - プロップスの説明を記載する
