# 03-03 ハイパーパラメータチューニング

## 概要
XGBoostのハイパーパラメータを最適化し、モデルの精度を向上させる。

## 詳細タスク

- [ ] ハイパーパラメータの探索範囲を定義する
  - max_depth: [4, 5, 6, 7, 8]
  - learning_rate: [0.01, 0.05, 0.1, 0.15]
  - n_estimators: [300, 500, 700, 1000]
  - subsample: [0.7, 0.8, 0.9]
  - colsample_bytree: [0.7, 0.8, 0.9]
  - min_child_weight: [1, 3, 5]
  - gamma: [0, 0.1, 0.2]
  - reg_alpha: [0, 0.1, 0.5]
  - reg_lambda: [1, 1.5, 2]

- [ ] グリッドサーチまたはベイズ最適化を実装する
  - notebooks/05_hyperparameter_tuning.ipynbを作成する
  - グリッドサーチを実装する（小規模な場合）
  - またはOptunaを使用したベイズ最適化を実装する
  - W&B Sweepsを使用する（推奨）

- [ ] W&B Sweepsを設定する
  - sweep設定ファイルを作成する
  - 探索方法を設定する（grid search, random search, bayesian等）
  - メトリクスを設定する（AUC-ROC等）
  - ハイパーパラメータの探索範囲を設定する

- [ ] 実験を実行する
  - 50〜100パターンの実験を実行する
  - 各実験をW&Bで記録する
  - 実験の進捗を確認する

- [ ] 実験結果を分析する
  - W&Bダッシュボードで結果を確認する
  - 最適なパラメータセットを特定する
  - パラメータの影響を分析する

- [ ] 最適パラメータでモデルを再学習する
  - 最適なパラメータセットを使用する
  - 8つのモデルを再学習する
  - 評価を実施する

- [ ] 最適パラメータをドキュメント化する
  - 最適なパラメータセットを記録する
  - README.mdまたはドキュメントに記載する
  - 再現性を確保する
