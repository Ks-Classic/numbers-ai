# 03-04 アンサンブル学習の検証

## 概要
複数のモデルを組み合わせたアンサンブル学習を実装し、精度向上を検証する。

## 詳細タスク

- [ ] アンサンブル手法を検討する
  - 投票法（Voting）
  - 重み付き平均法（Weighted Average）
  - スタッキング（Stacking）
  - 各手法のメリット・デメリットを比較する

- [ ] チャート・マスターモデルを構築する
  - 予測表ベースの特徴量のみで学習するモデルを構築する
  - 8つのモデルを作成する
  - モデルを評価する

- [ ] リハーサル・コンテキストモデルを構築する
  - リハーサル数字とコンテキスト情報のみで学習するモデルを構築する
  - 8つのモデルを作成する
  - モデルを評価する

- [ ] アンサンブル手法を実装する
  - notebooks/06_ensemble_learning.ipynbを作成する
  - 各モデルの予測結果を取得する処理を実装する
  - アンサンブル処理を実装する
  - 重みを調整する処理を実装する

- [ ] アンサンブルモデルを評価する
  - 単一モデルと比較する
  - AUC-ROCを計算する
  - Precision、Recall、F1-Scoreを計算する
  - W&Bで結果を記録する

- [ ] 最適なアンサンブル手法を決定する
  - 各手法の精度を比較する
  - 計算コストを比較する
  - 最適な手法を決定する

- [ ] アンサンブルモデルを実装する
  - 本番環境でアンサンブルモデルを使用するように更新する
  - APIを更新する
  - 動作確認をする
