# 03-02 特徴量エンジニアリング実験

## 概要
異なる特徴量セットを比較し、最適な特徴量セットを決定する。

## 詳細タスク

- [ ] 特徴量セットのパターンを定義する
  - パターン1: 形状特徴のみ
  - パターン2: 位置特徴のみ
  - パターン3: 関係性特徴のみ
  - パターン4: 形状+位置特徴
  - パターン5: 形状+位置+関係性特徴
  - パターン6: 全特徴量（既存）
  - その他、組み合わせを定義する

- [ ] 各特徴量セットで特徴量を計算する
  - notebooks/03_feature_engineering.ipynbを作成する
  - 各パターンで特徴量を計算する処理を実装する
  - 特徴量の次元数を確認する

- [ ] 各特徴量セットでモデルを学習する
  - 10パターンでモデルを学習する
  - W&Bで各実験を記録する
  - 学習時間を記録する

- [ ] モデル評価を実施する
  - 各パターンでAUC-ROCを計算する
  - Precision、Recall、F1-Scoreを計算する
  - Top-K Accuracyを計算する
  - W&Bダッシュボードで比較する

- [ ] 特徴量重要度を分析する
  - XGBoostのfeature_importanceを取得する
  - 重要度の高い特徴量を特定する
  - 不要な特徴量を特定する

- [ ] 最適な特徴量セットを決定する
  - 評価結果を比較する
  - 精度と計算コストのバランスを考慮する
  - 最適な特徴量セットを決定する

- [ ] 特徴量計算処理を更新する
  - 最適な特徴量セットを使用するように更新する
  - 不要な特徴量計算を削除する
  - パフォーマンスを最適化する
