# 04-03 マイクロサービス統合

## 概要
Next.jsフロントエンドとGCP Cloud RunのAIエンジンを統合し、マイクロサービスアーキテクチャを完成させる。

## 詳細タスク

- [ ] Cloud Run APIのエンドポイントを確認する
  - デプロイされたサービスのURLを確認する
  - エンドポイントのパスを確認する
  - 認証設定を確認する

- [ ] 環境変数を設定する
  - .env.localにCloud Run APIのURLを追加する
  - APIキーを設定する（必要に応じて）
  - CORS設定を確認する

- [ ] Next.jsからCloud Run APIを呼び出す処理を実装する
  - src/lib/api/cloud-run.tsを作成する
  - fetch関数を使用してAPIを呼び出す処理を実装する
  - エラーハンドリングを実装する
  - タイムアウト処理を実装する

- [ ] API Routesを更新する
  - src/app/api/predict/route.tsを更新する
  - Cloud Run APIを呼び出す処理に置き換える
  - レスポンスを変換する処理を実装する

- [ ] 認証・認可を実装する（必要に応じて）
  - APIキー認証を実装する
  - トークンベース認証を実装する（将来用）
  - セキュリティを強化する

- [ ] CORS設定を確認する
  - Cloud Run側でCORS設定を確認する
  - 許可するオリジンを設定する
  - 動作確認をする

- [ ] パフォーマンステストを実施する
  - API呼び出しのレスポンスタイムを測定する
  - 並列リクエストの処理を確認する
  - エラーレートを確認する

- [ ] モニタリング設定をする
  - Cloud Runのログを確認する
  - メトリクスを設定する
  - アラートを設定する（必要に応じて）

- [ ] エラーハンドリングを強化する
  - Cloud Run APIエラーの処理を実装する
  - タイムアウトエラーの処理を実装する
  - フォールバック処理を実装する（必要に応じて）

- [ ] 統合テストを実施する
  - エンドツーエンドのテストを実施する
  - 全機能が正常に動作することを確認する
  - パフォーマンスを確認する
