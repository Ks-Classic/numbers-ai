# 06-02 FastAPI AI推論エンジン実装

## 概要
FastAPIでAI推論エンジンを実装する。

## 詳細タスク

- [ ] FastAPIアプリケーションの基本構造を作成する
  - api/main.pyを作成する
  - FastAPIアプリケーションを初期化する
  - CORS設定を実装する

- [ ] ルーターを作成する
  - api/routers/predict.pyを作成する
  - /api/predictエンドポイントを実装する
  - リクエストボディの型定義を実装する（Pydantic）

- [ ] サービス層を作成する
  - api/services/chart_generator.pyを作成する
  - api/services/feature_extractor.pyを作成する
  - api/services/ai_predictor.pyを作成する

- [ ] モデル読み込み処理を実装する
  - モデルファイルを読み込む処理を実装する
  - モデルのキャッシュを実装する
  - モデル読み込みエラーの処理を実装する

- [ ] 特徴量計算処理を実装する
  - 予測表から特徴量を計算する処理を実装する
  - リハーサル数字から特徴量を計算する処理を実装する
  - 特徴量の正規化・スケーリングを実装する

- [ ] AI推論処理を実装する
  - 軸数字予測処理を実装する
  - 組み合わせ予測処理を実装する
  - スコア算出処理を実装する
  - ランキング処理を実装する

- [ ] レスポンス形式を実装する
  - 予測結果をフォーマットする処理を実装する
  - エラーレスポンスを実装する
  - レスポンスの型定義を実装する

- [ ] エラーハンドリングを実装する
  - バリデーションエラーの処理を実装する
  - モデルエラーの処理を実装する
  - その他の内部エラーの処理を実装する

- [ ] ロギングを実装する
  - リクエストログを記録する
  - エラーログを記録する
  - パフォーマンスログを記録する

- [ ] ヘルスチェックエンドポイントを実装する
  - /healthエンドポイントを実装する
  - モデルの状態を確認する処理を実装する
  - データベース接続を確認する処理を実装する（Phase 2以降）

- [ ] テストを実装する
  - ユニットテストを書く
  - 統合テストを書く
  - テストを実行して確認する
