# 技術スタック

## フロントエンド
- **Next.js 15.5.4+** (App Router)
- **React 19.1.0+**
- **TypeScript 5.3+**
- **Tailwind CSS 4.0+**
- **Zustand 5.0.8+** (状態管理)
