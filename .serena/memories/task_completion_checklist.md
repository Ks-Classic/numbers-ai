# タスク完了時のチェックリスト

## コード品質チェック
- [ ] TypeScriptの型エラーがないか確認（`pnpm build`で確認）
- [ ] ESLintエラーがないか確認（`pnpm lint`で確認）
- [ ] 不要なコメントやデバッグコードを削除
- [ ] 関数や変数に適切な名前が付けられているか確認

## テスト
- [ ] 関連するテストを実行して通過することを確認
  - `pnpm test:data-loader`（データローダー関連の場合）
  - `pnpm test:chart-generator`（チャート生成関連の場合）
  - `pnpm test:patterns`（4パターン関連の場合）
  - `pnpm test:api`（API関連の場合）

## 動作確認
- [ ] ローカル環境で動作確認
  - FastAPIサーバーが起動しているか確認
  - Next.js開発サーバーが起動しているか確認
  - ブラウザで実際に動作を確認

## ドキュメント
- [ ] 新規機能の場合は、必要に応じてドキュメントを更新
- [ ] 重要な変更は`docs/PROJECT_OVERVIEW.md`に反映
- [ ] API仕様の変更は`docs/design/03-data-api-design.md`に反映

## Git操作
- [ ] 変更をコミット（適切なコミットメッセージ）
- [ ] 必要に応じてリモートにプッシュ

## その他
- [ ] 環境変数（`.env.local`）が正しく設定されているか確認
- [ ] データファイルが正しく配置されているか確認（`data/`ディレクトリ）
- [ ] モデルファイルが存在するか確認（`data/models/`ディレクトリ）
