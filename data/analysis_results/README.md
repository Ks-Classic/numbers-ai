# 基礎集計結果データ

基礎集計の結果をカテゴリごとに整理したディレクトリです。

## ディレクトリ構造

- `{集計項目名}_{集計範囲}.json`

**集計範囲の名称:**
- `全範囲`: 1-6850回（全期間）
- `週5実施開始以降`: 1340-6850回（週5実施開始から最新まで）
- `リハーサル導入以降`: 4801-6850回（リハーサル数字が有効な範囲）

## 各カテゴリの説明

### 01_keisen_base_stats: keisen基礎集計

**ファイル一覧:**
- `パターン出現頻度_{範囲}.json`: 各桁 × 前々回(0-9) × 前回(0-9)の組み合わせごとの出現頻度
- `当選数字ランキング_{範囲}.json`: 各パターンごとの当選数字のランキング
- `統計的信頼性指標_{範囲}.json`: サンプル数、標準偏差、95%信頼区間、信頼性スコア

**生成スクリプト:**
- `scripts/base_statistics/01_keisen_base_stats/analyze_keisen_base_stats.py`

### 02_keisen_comparison: 旧keisen vs 新keisen比較

**実装予定**: 旧keisen（1340-6391回）と新keisen（4801-6850回）で生成したCUBEを比較分析した結果。

### 03_temporal_analysis: 時系列・周期性分析

**実装予定**: 曜日、月相（月齢）、季節、月、年・四半期などの時間軸で当選番号の出現パターンを分析した結果。

### 04_digit_patterns: 数値パターン分析

**実装予定**: 各数字（0-9）の出現頻度、よく出現する数字の組み合わせ、連続性・周期性を分析した結果。

### 05_cube_characteristics: CUBE特性分析

**実装予定**: CUBEのサイズ、密度、メイン行の特性を分析した結果。

### 06_correlation: 相関分析

**実装予定**: リハーサル数字と当選番号、N3とN4、前回・前々回との相関を分析した結果。

### 07_anomaly_detection: 異常値・外れ値分析

**実装予定**: CUBE内に出現しなかった回、リハーサル数字と当選番号が全く重ならなかった回などの異常パターンを検出した結果。

### 08_extreme_cube: 極CUBE基礎集計

**実装予定**: 極CUBE内での当選番号出現分析、数字の出現パターン、時系列・周期性分析を実施した結果。
