# 分析レポート (Analysis Reports)

## 概要

このディレクトリには、データ分析の結果をまとめた人間向けレポートが格納されています。

## ディレクトリ構造

```
data/reports/
├── 00_common_analysis_report.md      # 共通分析まとめレポート
├── 01_cube_analysis_report.md        # 通常CUBE分析まとめレポート
├── 02_extreme_cube_analysis_report.md # 極CUBE分析まとめレポート
├── model_improvement/                 # モデル改善関連レポート
│   ├── ハイパーパラメータチューニング結果レポート.md
│   ├── モデル改善統合評価レポート.md
│   ├── モデル改良戦略の判断材料.md
│   ├── 予測モデル詳細分析レポート.md
│   ├── 特徴量エンジニアリング改善-実装完了報告.md
│   └── 特徴量エンジニアリング改善ログ.md
└── assets/                            # レポート用の画像・グラフ
    ├── feature_importance_n3_axis.png
    ├── feature_importance_n4_axis.png
    └── category_importance_*.png
```

## レポートの種類

### カテゴリ別分析レポート

各分析カテゴリの結果をストーリー性を持ってまとめたレポート。

#### 00_common_analysis_report.md
- **内容**: 共通分析（数字出現パターン、時系列周期性、異常値検出など）の統合レポート
- **対象読者**: データサイエンティスト、プロジェクトメンバー
- **更新タイミング**: 共通分析の各タスク完了時

#### 01_cube_analysis_report.md
- **内容**: 通常CUBE分析（keisen基礎集計、新旧比較、CUBE特性、相関分析など）の統合レポート
- **対象読者**: データサイエンティスト、プロジェクトメンバー
- **更新タイミング**: 通常CUBE分析の各タスク完了時

#### 02_extreme_cube_analysis_report.md
- **内容**: 極CUBE分析（全期間基礎集計、並び型分析、区切り別比較など）の統合レポート
- **対象読者**: データサイエンティスト、プロジェクトメンバー
- **更新タイミング**: 極CUBE分析の各タスク完了時

### モデル改善レポート

予測モデルの改善に関する詳細レポート。

- **ハイパーパラメータチューニング結果レポート.md**: パラメータ最適化の結果
- **モデル改善統合評価レポート.md**: 複数の改善施策の統合評価
- **予測モデル詳細分析レポート.md**: モデルの詳細な性能分析

## レポートと生データの関係

```
分析実行
  ↓
data/analysis_results/     # 生データ・集計結果（JSON、CSV）
  ↓                        # スクリプトが自動出力
分析完了・考察
  ↓
data/reports/              # 人間向けレポート（Markdown）
                           # 手動でまとめる
```

**使い分け**:
- `analysis_results/`: 機械可読、再利用可能、自動処理向け
- `reports/`: 人間可読、ストーリー性、意思決定向け
